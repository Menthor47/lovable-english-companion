# AGSEO: AI-Driven SEO Agency

The intelligent companion for modern marketing teams.

## Tech Stack

* React + TypeScript + Vite
* Shadcn UI + Tailwind CSS
* Framer Motion
* React Router DOM
* Recharts

## Features

* **AI-First Services**: Content Engine, GEO Optimization, Technical Audits.
* **Interactive Tools**: ROI Calculator, Instant Audit.
* **Client Operations**: Integrated Client Portal Dashboard.
* **Programmatic SEO**: Industry-specific landing pages and comparison hubs.

## Prerequisites

* Node.js (v18 or higher)
* npm (v9 or higher)

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file:

- `VITE_FIREBASE_API_KEY`, `VITE_FIREBASE_AUTH_DOMAIN`, `VITE_FIREBASE_PROJECT_ID`
- `VITE_FIREBASE_STORAGE_BUCKET`, `VITE_FIREBASE_MESSAGING_SENDER_ID`, `VITE_FIREBASE_APP_ID`
- `VITE_SITE_URL`, `VITE_CONTACT_EMAIL`, `VITE_WHATSAPP_NUMBER`
- `VITE_SENTRY_DSN` (optional)

## Getting Started

1. Clone the repository
2. Install dependencies:

    ```sh
    npm install
    ```

3. Set up environment variables (copy .env.example to .env)
4. Start the development server:

    ```sh
    npm run dev
    ```

## Testing

Run tests with:

```sh
npm test
```
